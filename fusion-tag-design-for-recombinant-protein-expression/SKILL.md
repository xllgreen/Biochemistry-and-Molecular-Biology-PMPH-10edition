# 融合标签设计用于重组蛋白表达

## 何时使用
- 表达目的是制备抗体或简化纯化流程
- 已选定表达系统（如大肠杆菌）
- 目的基因已完成克隆
- 下游应用对蛋白结构或活性有特定要求（如需天然序列）

## 执行步骤
1. **选择合适的标签系统**：根据纯化需求和下游用途，从以下常用标签中选择：
   - GST（谷胱甘肽-S-转移酶）：高亲和力，适合一步亲和纯化
   - His-tag（6×His）：小分子标签，适用于多数表达系统
   - FLAG 或 MBP（麦芽糖结合蛋白）：用于特定检测或提高可溶性

2. **构建融合基因载体**：将所选标签编码序列融合至目的基因的5'端或3'端。

3. **插入可切割位点**：在标签与目的蛋白之间必须引入特异性蛋白酶识别位点，例如：
   - 凝血酶位点（LVPR↓GS）
   - 因子Xa位点（IEGR↓）
   - TEV蛋白酶位点（ENLYFQ↓S）

4. **执行亲和纯化**：利用标签特性进行一步纯化（如GST标签使用谷胱甘肽-Sepharose树脂）。

5. **评估是否去除标签**：若最终用途要求完整生物学功能（如酶活性、受体结合），则在纯化后使用对应蛋白酶切除标签。

6. **处理包涵体情况**：若表达产物为包涵体，先进行变性-复性，再切除标签并验证功能。

## 注意事项
- 即使是小标签（如His-tag）也可能干扰蛋白折叠、稳定性或抗原性。
- 标签切除会增加实验步骤、成本和潜在蛋白降解风险，需权衡利弊。
- 裂解效率受位点可及性影响，建议在设计阶段预测空间构象。
