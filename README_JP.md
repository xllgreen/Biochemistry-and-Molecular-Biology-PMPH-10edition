## 概要

本プロジェクトは生化学と分子生物学の核心的知識体系を体系的に統合し、**154の主要スキル**を16カテゴリーに分類しています。

**対象読者**：医学生、生物医学研究者、臨床医、生化学教育者、薬学・生命科学専門家

**参考教材**：人民衛生出版社『生化学と分子生物学』第10版（ISBN: 978-7-117-36618-8）

**⚠️ リスク ⚠️**：本スキル集は教材内容を学習参考用に整理したものです。臨床診断や治療に関する決定は、資格を持つ医療専門家が行う必要があります。

## プロジェクト構造

```
Biochemistry-MolecularBiology-PMPH-10edition/
├── SKILL.md                    # コア設定 — 154スキルのレジストリ
├── README.md                   # ドキュメント（中国語）

├── README_FR.md                # ドキュメント（フランス語）
├── README_RU.md                # ドキュメント（ロシア語）

└── assets/                     # 静的リソース
```

## 使用例

```bash
# 概念説明
糖代謝におけるPFK-1の調節メカニズムを説明してください

# 臨床ケース分析
LDL受容体欠損による家族性高コレステロール血症の分子メカニズムを分析してください

# 実験設計
大腸菌でのヒトリコンビナントタンパク質発現プロトコルを設計してください
```
