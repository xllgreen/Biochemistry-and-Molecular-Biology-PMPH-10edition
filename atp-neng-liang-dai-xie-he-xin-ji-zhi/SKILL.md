# ATP能量代谢核心机制

## 触发条件
当细胞需要直接可用的能量时，启动ATP水解或再生机制。

## 执行步骤

1. **识别ATP供能方式**：根据细胞需求选择以下任一机制：
   - **直接水解**：ATP → ADP + Pi，释放能量驱动吸能反应（如离子泵、肌肉收缩）；
   - **基团转移**：将Pi、PPi或AMP共价连接到底物（如葡萄糖→葡萄糖-6-磷酸），活化底物；
   - **核苷酸互变**：通过核苷二磷酸激酶催化，利用ATP再生UTP、CTP或GTP，用于糖原、磷脂或蛋白质合成。

2. **调节ATP/ADP稳态**：通过腺苷酸激酶催化反应 2ADP ↔ ATP + AMP，响应细胞能量状态变化。

3. **调用磷酸肌酸储备系统**：在骨骼肌、心肌或脑组织中，当ATP快速消耗时，利用磷酸肌酸（CP）再生ATP：
   - ATP充足时：ATP + 肌酸 → ADP + CP；
   - ATP不足时：CP + ADP → ATP + 肌酸。

4. **确认适用条件**：确保存在Mg²⁺和相关激酶（腺苷酸激酶、核苷二磷酸激酶、肌酸激酶），且非长期厌氧供能场景。

## 输出结果
成功驱动耗能反应、活化代谢底物或再生其他核苷三磷酸（UTP/CTP/GTP）。
