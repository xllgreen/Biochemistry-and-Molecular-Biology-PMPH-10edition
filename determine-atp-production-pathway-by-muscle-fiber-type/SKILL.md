# 肌纤维类型与ATP生成主导途径判定

## 何时使用
- 用户询问不同类型肌纤维的能量代谢差异
- 需要为耐力或爆发性训练制定科学依据
- 分析运动中乳酸积累或疲劳原因
- 解释为何某些运动依赖脂肪而非糖原供能

## 执行步骤

1. **确认肌纤维类型**：
   - 若为Ⅰ型（慢肌/红肌），进入步骤2；
   - 若为Ⅱ型（快肌/白肌，含Ⅱa/Ⅱb亚型），进入步骤3。

2. **Ⅰ型肌纤维处理**：
   - 主导ATP生成途径：氧化磷酸化；
   - 主要底物：脂肪酸（首选）、酮体、肌糖原、血糖；
   - 终产物：CO₂ + H₂O，无乳酸积累；
   - 运动表现：抗疲劳性强，适用于长时间低强度活动（如长跑、姿势维持）。

3. **Ⅱ型肌纤维处理**：
   - 主导ATP生成途径：糖酵解中的底物水平磷酸化；
   - 主要底物：肌糖原（经糖原磷酸化酶分解为葡萄糖-6-磷酸）；
   - 终产物：乳酸（由乳酸脱氢酶催化丙酮酸还原生成）；
   - 运动表现：收缩快速有力但易疲劳，适用于短时高强度爆发（如冲刺、举重）。

4. **验证约束条件**：
   - 确保对象为骨骼肌组织；
   - 注意所有骨骼肌均缺乏葡萄糖-6-磷酸酶，无法进行糖异生。

5. **输出结果**：
   - 明确说明肌纤维类型对应的供能系统、底物偏好、抗疲劳能力及适用运动场景，用于运动处方设计或代谢干预建议。
