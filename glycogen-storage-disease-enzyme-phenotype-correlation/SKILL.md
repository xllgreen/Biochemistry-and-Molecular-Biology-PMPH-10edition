# 糖原贮积症的酶缺陷与临床表型关联分析

## 何时使用
- 患者出现肝大、空腹低血糖、乳酸血症、高脂血症、心肌病或运动不耐受等症状
- 临床怀疑为遗传性糖原代谢障碍
- 需要根据表型初步判断糖原贮积症类型以指导进一步酶学或基因检测

## 执行步骤

1. **基于主要临床表现初步分型**：
   - **严重低血糖 + 肝大 + 乳酸血症 + 高脂血症** → 考虑 **I型**（葡萄糖-6-磷酸酶缺陷）
   - **婴儿期心肌肥厚、全身多器官受累、早期死亡** → 考虑 **II型**（溶酶体α-葡萄糖苷酶缺陷）
   - **仅肝大、无严重代谢紊乱** → 考虑 **VI型**（肝磷酸化酶缺陷）或 **VIII型**（肝磷酸化酶激酶缺陷）
   - **运动后肌痛、肌红蛋白尿、肌无力（无肝受累）** → 考虑 **V型**（肌磷酸化酶缺陷）或 **VII型**（肌磷酸果糖激酶-1缺陷）

2. **结合酶缺陷与器官受累模式确认分型**：
   - **I型**：缺陷酶为葡萄糖-6-磷酸酶；受累器官为肝、肾；糖原结构正常
   - **II型**：缺陷酶为溶酶体α-葡萄糖苷酶；受累器官为全身组织（尤其心肌）；糖原结构正常
   - **III型**：缺陷酶为脱支酶；受累器官为肝、肌肉；糖原分支多、外周糖链短
   - **IV型**：缺陷酶为分支酶；受累器官为肝、肌肉、脾；糖原分支少、外周糖链特别长

3. **解释关键病理机制以支持诊断**：
   - 葡萄糖-6-磷酸酶缺失 → 无法释放葡萄糖入血 → 严重空腹低血糖
   - 溶酶体α-葡萄糖苷酶缺失 → 无法降解糖原 → 全身糖原堆积 → 心肌功能衰竭
   - 肝磷酸化酶缺失 → 肝糖原分解障碍 → 肝大但代谢相对稳定

4. **输出结果**：返回糖原贮积症的具体分型（I–VIII型）、对应缺陷酶名称、主要受累器官列表、糖原结构特征及简要预后评估。

> 注意：本技能仅适用于先天性酶缺陷导致的糖原贮积症，不适用于获得性糖原代谢异常或非酶缺陷引起的糖原累积。
