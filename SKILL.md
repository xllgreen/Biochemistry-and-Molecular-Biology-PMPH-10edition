# 《Biochemistry & Molecular Biology (PMPH 10th Edition)》Skill Collection

> **基于人民卫生出版社《生物化学与分子生物学》第10版的AI技能合集**

本技能合集源自权威教材，涵盖 **154 项核心技能**，分为分子生物学基础、基因表达调控、代谢通路与调控、信号转导、基因组学与精准医学、临床生化与诊断等15大分类。

## 使用场景 / When to Use

本技能合集适用于以下场景：

### 教学与学习
- 医学生/研究生系统学习生物化学与分子生物学
- 教师备课、课堂互动与考试出题
- 解释代谢通路、信号转导等复杂机制

### 科研辅助
- 实验设计方案（PCR、克隆、表达、纯化）
- 代谢组学/基因组学分析策略
- 疾病相关基因功能分析

### 临床参考
- 代谢性疾病诊断与鉴别
- 精准医学与靶向治疗
- 临床生化检验结果解读

## 执行规则 / Execution Rules

### 技能调用

用户直接引用技能名称可触发对应专业知识。例如：

- "请解释 **fructose-metabolism-pathways-and-pathological-risks** 的关键酶"
- "基于 **precision-medicine-four-layer-system** 设计方案"
- "分析 **fatty-acid-beta-oxidation-and-energy-yield-calculation** 的ATP产量"

### 响应规则

1. **结构化输出** — 每项技能输出包含：机制/原理、步骤/流程、临床/应用意义
2. **教材引用** — 优先引用第10版教材内容，标注页码/章节
3. **双语术语** — 核心术语首次出现时标注英文
4. **风险评估** — 涉及临床诊断/用药的技能须标注风险提示
