# RNA生物合成的模板依赖方式

## 何时使用
- 分析细胞内RNA生成过程（如mRNA、rRNA等的合成）
- 解释RNA病毒如何在宿主细胞中复制其基因组
- 区分转录与RNA复制的酶学基础、模板来源及生物学功能
- 回答涉及RNA聚合酶类型或模板依赖性的分子生物学问题

## 执行步骤

### 1. 判断模板类型
- 若模板为**DNA** → 属于**DNA依赖的RNA合成（转录）**
- 若模板为**RNA** → 属于**RNA依赖的RNA合成（RNA复制）**

### 2. 确定生物背景
- **细胞环境**（真核或原核）：仅发生**转录**，由DNA指导RNA合成
- **RNA病毒感染的宿主细胞**：可能发生**RNA复制**，用于病毒基因组扩增

### 3. 识别关键特征
- **共同点**：
  - 均由依赖模板的聚合酶催化
  - 合成方向均为5'→3'
  - 使用NTP（ATP、UTP、CTP、GTP）作为底物
  - 形成3',5'-磷酸二酯键
- **差异点**：
  - **转录**：模板为DNA的一条链，产物包括mRNA/rRNA/tRNA等，由**DNA依赖的RNA聚合酶**催化，无需引物
  - **RNA复制**：模板为病毒单链RNA，产物为互补RNA或基因组拷贝，由**RNA依赖的RNA聚合酶**（RdRp）催化

### 4. 输出结论
根据模板类型、聚合酶种类和生物背景，明确RNA合成方式及其生物学意义：
- 转录 → 支持细胞正常基因表达
- RNA复制 → 支持RNA病毒增殖
