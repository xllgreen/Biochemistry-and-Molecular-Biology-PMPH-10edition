# 第二信使的生成、作用与清除机制

## 何时使用
- 细胞外配体与G蛋白偶联受体或受体酪氨酸激酶结合后
- 需要解释信号如何从膜表面传递至胞内效应器
- 分析信号终止或持续性的分子基础

## 执行步骤

1. **第二信使生成**：确认上游酶是否被激活，并识别对应的第二信使：
   - 腺苷酸环化酶（AC）→ cAMP
   - 鸟苷酸环化酶（GC）→ cGMP
   - 磷脂酶C（PLC）→ DAG + IP₃
   - 内质网释放 → Ca²⁺浓度升高
   - cGAS → cGAMP（特例，激活STING通路）
   - NO可直接激活GC

2. **下游激活**：第二信使作为别构激活剂结合并激活效应蛋白：
   - cAMP → 激活PKA
   - cGMP → 激活PKG
   - DAG + Ca²⁺ → 激活PKC
   - IP₃ → 促进内质网释放Ca²⁺
   - Ca²⁺ → 结合钙调蛋白等

3. **信号清除**：识别终止机制以恢复基线状态：
   - 磷酸二酯酶（PDE）水解cAMP/cGMP
   - Ca²⁺-ATP酶将Ca²⁺泵回内质网或排出胞外
   - DAG被激酶磷酸化或水解

4. **判断信号持续性**：若上游酶持续激活，则第二信使水平维持；否则信号自动终止。

> 注意：本机制不适用于无第二信使参与的直接蛋白-蛋白相互作用通路（如JAK-STAT的部分环节）。
