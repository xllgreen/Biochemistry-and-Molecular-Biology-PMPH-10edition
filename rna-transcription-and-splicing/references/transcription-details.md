# 转录机制详细说明

## 原核 vs 真核转录对比

| 特征 | 原核生物 | 真核生物 |
|------|--------|--------|
| RNA聚合酶 | 单一类型（含σ因子） | Pol I（rRNA）、Pol II（mRNA）、Pol III（tRNA等） |
| 启动子核心元件 | Pribnow盒（-10区：TATAAT） | TATA盒（-25至-30区）、CAAT盒、GC盒等 |
| 转录与翻译偶联 | 是 | 否（空间分离） |
| mRNA加工 | 通常无 | 5'帽、3' polyA尾、剪接 |

## α-鹅膏草碱敏感性
- Pol I：不敏感
- Pol II：高度敏感（IC50 ≈ 1 μg/mL）
- Pol III：中度敏感（IC50 ≈ 10–20 μg/mL）

## 转录终止机制
- **原核ρ依赖型**：ρ因子结合rut位点，解旋RNA-DNA杂交体
- **原核内在终止**：富含GC的发夹结构 + 连续U序列
- **真核终止**：与polyA信号（AAUAAA）相关，切割后加尾触发Pol II脱离
