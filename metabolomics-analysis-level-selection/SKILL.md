# 代谢组学分析层次选择

## 何时使用
- 需要对生物体系中的代谢产物进行系统性分析
- 已明确研究目标（如寻找疾病标志物、研究特定代谢通路、全面评估代谢状态或快速筛查样本差异）
- 已确定分析范围为靶向或全局代谢物分析

## 如何执行
1. **确认适用对象**：确保分析对象为生物体液、组织样品、细胞培养液或相关代谢物分析任务。
2. **排除不适用场景**：本技能不适用于基因组、蛋白质组分析，也不涵盖非小分子代谢物。
3. **根据研究目的选择分析层次**：
   - **若关注特定疾病标志物或已知代谢物** → 选择 **代谢物靶标分析**（仅对一个或几个特定代谢组分进行定性或定量）
   - **若聚焦某类结构相关化合物或特定代谢通路**（如糖酵解）→ 选择 **代谢谱分析**（对预设目标代谢物组进行定量）
   - **若需全面了解生物体系的整体代谢状态** → 选择 **代谢组学**（对所有小分子代谢物进行定性和定量）
   - **若需快速比较多个样本的总体差异**（如药物响应筛查）→ 选择 **代谢指纹分析**（高通量整体定性，不分离单一代谢物）
4. **输出所选层次及其适用场景说明**。

> 注意：选择前必须明确研究目标和分析范围（靶向 vs 全局），否则可能导致方法误用。
