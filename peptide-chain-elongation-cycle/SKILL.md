# 肽链延长三步循环机制

## 触发条件
当核糖体A位对应mRNA上的**非终止密码子**，且翻译起始复合物已形成时，启动本循环。

## 执行步骤

### 1. 进位（注册）
- 氨酰tRNA与GTP-EF-Tu（原核）或GTP-eEF1α（真核）形成三元复合物；
- 该复合物进入核糖体A位，反密码子与mRNA密码子配对；
- 正确配对触发GTP水解，EF-Tu-GDP释放；错误配对因无法稳定结合而解离（校正机制）。

### 2. 成肽
- P位的肽酰-tRNA（或起始fMet-tRNA）与A位氨酰-tRNA的α-氨基发生缩合反应，形成新肽键；
- 反应由**肽酰转移酶**催化，该酶为rRNA构成的核酶（原核：23S rRNA；真核：28S rRNA）；
- 成肽后，新生肽链转移至A位tRNA，P位留下卸载的tRNA。

### 3. 转位
- EF-G（原核）或eEF2（真核）结合GTP，发挥转位酶活性；
- GTP水解供能，驱动核糖体沿mRNA向3'端移动一个密码子；
- 结果：
  - A位的肽酰-tRNA移至P位；
  - P位卸载tRNA移至E位并脱落；
  - A位空出，对应下一密码子。

### 4. 能量核算
- 每轮循环水解**2分子GTP**（进位1 + 转位1）；
- 加上氨基酸活化消耗的2 ATP（等价于2高能磷酸键），每形成一个肽键至少消耗**4个高能磷酸键**。

### 5. 循环终止
- 重复上述三步，直至A位遇到**终止密码子**，此时不适用本机制，应切换至翻译终止流程。

## 关键变量
- **EF-Tu / eEF1α**：介导氨酰tRNA进位，依赖GTP结合与水解；
- **EF-G / eEF2**：驱动转位，具GTP酶活性；
- **肽酰转移酶**：rRNA核酶，催化肽键形成；
- **GTP**：每轮延长循环消耗2分子。

## 输出结果
肽链C端增加一个氨基酸残基，核糖体准备读取下一密码子。
