# PFK-1别构与激素调控机制

## 何时使用
- 用户询问糖酵解关键酶PFK-1的调控机制
- 需要解释ATP/AMP比值变化对糖酵解的影响
- 涉及胰岛素或胰高血糖素如何调节肝脏糖代谢
- 分析肌肉与肝脏在能量代谢调控中的差异

## 执行步骤

1. **识别调控输入**：
   - 检测细胞内ATP、AMP、ADP、柠檬酸浓度变化
   - 判断是否存在胰岛素或胰高血糖素信号
   - 确认组织类型（肝 vs 肌肉）

2. **应用别构调节规则**：
   - 若ATP浓度高 → 抑制PFK-1活性
   - 若AMP或ADP浓度高 → 激活PFK-1，抵消ATP抑制
   - 若柠檬酸浓度高 → 增强ATP的抑制作用

3. **整合激素信号通路**：
   - 胰高血糖素升高 → 激活PKA → 双功能酶磷酸化 → F-2,6-BP水平下降 → PFK-1抑制
   - 胰岛素升高 → 激活磷酸酶 → 双功能酶去磷酸化 → F-2,6-BP水平上升 → PFK-1激活

4. **输出调控结论**：
   - 综合能量状态与激素信号，判断糖酵解是加速还是受抑
   - 区分组织特异性响应：肌肉侧重AMP/ATP比值，肝脏侧重激素调控

> 注意：果糖-2,6-二磷酸（F-2,6-BP）是PFK-1最强激活剂，其水平由双功能酶（PFK-2/FBP-2）动态控制。
