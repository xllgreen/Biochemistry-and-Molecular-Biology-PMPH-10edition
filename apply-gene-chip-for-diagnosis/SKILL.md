# 基因芯片用于大规模基因诊断

## 何时使用
- 需同时检测多个已知基因/位点的表达水平、拷贝数变异或甲基化状态
- 样本为组织或细胞来源的DNA/RNA
- 目标疾病包括单基因遗传病（如地中海贫血）、肿瘤或已知病原体感染
- 具备基因芯片平台和探针设计能力

## 执行步骤
1. **芯片制备**：将已知序列的核酸探针（cDNA或寡核苷酸）高密度固定于固相载体（如玻片）。
2. **样本处理**：从待测样本提取总RNA或DNA；若为RNA，需逆转录并荧光标记。
3. **杂交反应**：将标记样本与芯片探针在严格控制条件下杂交，依赖碱基互补配对结合。
4. **信号检测**：洗脱未结合片段后，用激光扫描仪读取各探针位点的荧光信号强度。
5. **结果解读**：
   - 信号强度反映目标基因的表达量、拷贝数或甲基化程度；
   - 应用于单基因病筛查、肿瘤表达谱分析、CGH、病原体耐药突变检测（如乙肝YMDD突变）。
6. **注意事项**：
   - 仅适用于已知序列目标，无法发现全新突变；
   - 结果需结合临床信息综合判断。

> 优势：高通量、自动化、微量样本、高灵敏度与特异性。
