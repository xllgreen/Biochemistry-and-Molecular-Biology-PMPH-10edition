# 组织特异性甘油三酯合成途径

## 触发条件
- 细胞类型为小肠黏膜细胞、肝细胞或脂肪细胞
- 存在活化的脂肪酸CoA和甘油骨架（甘油一酯或甘油-3-磷酸）

## 执行步骤

### 1. 脂肪酸活化（所有组织共通）
将脂肪酸活化为脂肪酰CoA：
```
脂肪酸 + CoA-SH + ATP → 脂肪酰CoA + AMP + PPi
```
由脂肪酰CoA合成酶催化。

### 2. 小肠黏膜细胞：甘油一酯途径
- 利用膳食消化产生的2-甘油一酯
- 在脂肪酰CoA转移酶作用下，依次将两分子脂肪酰CoA加至2-甘油一酯的1位和3位羟基
- 直接生成甘油三酯，无需从头合成甘油骨架

### 3. 肝细胞与脂肪细胞：甘油二酯途径
- 起始物为糖酵解中间产物磷酸二羟丙酮还原生成的甘油-3-磷酸
- 甘油-3-磷酸先后与两分子脂肪酰CoA酯化，形成1,2-甘油二酯
- 1,2-甘油二酯再与第三分子脂肪酰CoA酯化，生成甘油三酯

### 4. 关键限制条件
- **脂肪细胞**缺乏甘油激酶，不能直接利用游离甘油合成甘油三酯
- **肝细胞**不能储存甘油三酯，必须将其组装为VLDL分泌入血
- **肝、肾等组织**含甘油激酶，可将游离甘油磷酸化为甘油-3-磷酸用于合成

## 输出结果
明确指定目标组织所采用的甘油三酯合成路径，并说明其对脂质代谢效率、储存方式及底物利用的影响。
