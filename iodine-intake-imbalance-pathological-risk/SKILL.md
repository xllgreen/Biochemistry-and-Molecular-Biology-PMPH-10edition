# 碘摄入失衡病理风险评估

## 何时使用
- 用户描述长期膳食碘摄入异常（明显低于或高于推荐量）
- 出现甲状腺肿大、智力发育迟缓、心悸、多汗等症状
- 涉及孕妇、婴幼儿或儿童的碘营养咨询
- 需判断是否因碘摄入失衡导致内分泌或发育问题

## 执行步骤

1. **判断碘摄入状态**：
   - 若长期碘摄入 < 生理需要量（成人约150 μg/d），视为缺碘
   - 若长期碘摄入显著 > 安全上限（通常 > 1100 μg/d），视为碘过量

2. **评估缺碘后果**：
   - 轻中度缺碘 → 触发甲状腺代偿性增生 → 可能发展为地方性甲状腺肿
   - 严重缺碘（尤其在胎儿期或婴幼儿期）→ 甲状腺激素合成不足 → 中枢神经系统发育障碍 → 表现为智力低下、发育停滞，胎儿期可致“呆小病”（克汀病）

3. **评估碘过量后果**：
   - 碘过量 → 抑制甲状腺激素合成（Wolff-Chaikoff效应）→ TSH反馈性升高 → 甲状腺滤泡增生 → 高碘性甲状腺肿
   - 可伴甲亢症状（心悸、多汗）或碘中毒表现（恶心、金属味觉）

4. **输出建议**：
   - 提示存在碘缺乏或过量风险
   - 建议调整膳食碘来源（如加碘盐、海产品）
   - 推荐医学检查（如尿碘、TSH、甲状腺超声）及专业干预

> 注意：短期碘摄入波动通常不引发明显病理变化，仅长期失衡需干预。
