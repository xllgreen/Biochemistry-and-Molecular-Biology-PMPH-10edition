# 嘧啶核苷酸分解代谢产物分析

## 何时使用
- 需评估嘧啶核苷酸代谢是否导致尿路结晶或痛风样症状
- 解释临床检测中尿液β-氨基异丁酸水平升高的原因（如肿瘤患者接受放化疗后）
- 区分嘌呤与嘧啶核苷酸代谢终产物特性
- 分析肝脏功能对核苷酸代谢的影响

## 执行步骤
1. **确认底物类型**：仅适用于嘧啶核苷酸（胞嘧啶、尿嘧啶、胸腺嘧啶、dTMP），不适用于嘌呤核苷酸。
2. **验证前提条件**：确保存在核苷酸酶、核苷磷酸化酶且肝脏功能正常。
3. **执行分解流程**：
   - 嘧啶核苷酸 →（核苷酸酶）→ 嘧啶核苷
   - 嘧啶核苷 →（核苷磷酸化酶）→ 游离嘧啶碱基 + 核糖-1-磷酸
4. **追踪碱基特异性路径**：
   - 胞嘧啶 →（脱氨基）→ 尿嘧啶
   - 尿嘧啶 →（还原→水解开环）→ NH₃ + CO₂ + β-丙氨酸
   - 胸腺嘧啶 → β-氨基异丁酸
5. **评估终产物特性**：所有产物（NH₃、CO₂、β-丙氨酸、β-氨基异丁酸）均为水溶性小分子，无结晶风险。
6. **临床关联判断**：若患者尿中β-氨基异丁酸显著升高，提示DNA大量分解（如肿瘤治疗、高DNA饮食）。

> 注意：此过程主要在肝脏进行，肝功能异常可能影响代谢效率。
