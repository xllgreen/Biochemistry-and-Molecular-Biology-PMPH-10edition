# 原癌基因在生长因子信号通路中的功能分类

## 适用条件
- 分析对象为已知或疑似参与生长因子信号通路的原癌基因产物
- 细胞处于增殖或分化调控状态
- 存在原癌基因表达异常或突变证据
- **不适用于**抑癌基因产物

## 执行步骤
1. **确定蛋白亚细胞定位**：根据实验数据或数据库注释判断蛋白位于细胞外、跨膜、胞内或核内。
2. **匹配功能类别**：依据定位与已知功能，归入以下四类之一：
   - **细胞外生长因子**：如SIS（PDGF-β）、INT-2（FGF同类物），通过结合膜受体激活下游通路（MAPK、PLC-γ/DAG-PKC）。
   - **跨膜生长因子受体**：如EGFR、HER2、KIT，具酪氨酸激酶活性，接收胞外信号。
   - **细胞内信号转导分子**：如RAS、RAF、SRC/ABL，介导信号向核内传递。
   - **核内转录因子**：如MYC、FOS/JUN（AP-1复合物），直接调控增殖相关基因转录。
3. **关联致癌机制**：说明该蛋白因突变或过表达如何导致信号持续激活，引发细胞增殖失控。
4. **输出结果**：明确功能类别、代表基因、作用机制及对靶向治疗的启示。

> 注：所有分类均以“突变或过表达”为癌变前提。
