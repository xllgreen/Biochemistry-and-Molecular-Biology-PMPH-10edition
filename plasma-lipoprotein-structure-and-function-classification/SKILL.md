# 血浆脂蛋白的结构与功能分类

## 何时使用
- 解释脂质如何在水相血浆环境中被运输
- 分析不同脂蛋白（CM、VLDL、LDL、HDL等）的结构与功能关系
- 理解载脂蛋白（如apo B48、apo B100、apo CII、apo E）如何决定脂蛋白代谢命运
- 评估临床血脂异常背后的病理生理机制

## 执行步骤

1. **确认脂蛋白基本结构**：所有血浆脂蛋白均由疏水内核（甘油三酯TG和胆固醇酯CE）和亲水外层（载脂蛋白、磷脂、游离胆固醇）构成，确保颗粒在血浆中稳定可溶。

2. **识别内核成分差异**：
   - CM 和 VLDL：以内源性或外源性 TG 为主（CM含80–95% TG，VLDL含50–70%）
   - LDL 和 HDL：以 CE 为主（LDL含40–42% CE，HDL含15–17%）

3. **分析载脂蛋白功能**：
   - apo B48：仅存在于CM，由小肠合成，促进CM组装
   - apo B100：存在于VLDL/LDL，介导LDL受体识别
   - apo CII：激活脂蛋白脂肪酶（LPL），显著增强TG水解
   - apo E：参与残粒清除，介导肝细胞受体结合
   - apo A I：HDL主要蛋白，激活LCAT促进胆固醇酯化

4. **关联结构与功能**：
   - CM：转运饮食来源的TG和胆固醇
   - VLDL：转运肝脏合成的内源性TG
   - LDL：将胆固醇输送至外周组织
   - HDL：执行胆固醇逆向转运回肝脏

5. **验证适用条件**：仅适用于血浆中脂质运输场景，不适用于细胞内脂滴储存形式。
