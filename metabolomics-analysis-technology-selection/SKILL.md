# 代谢组学分析技术选择

## 触发条件
当需对复杂生物样本（如代谢物样本、生物体液、组织提取液）中的代谢物进行分离、定性或定量分析，且已完成样本预处理并明确检测目标（定性/定量）时，应用本技能。

## 执行流程

1. **评估代谢物性质与分析需求**
   - 确定代谢物的理化性质（analyte_properties）：极性、挥发性、热稳定性等
   - 明确所需输出类型（required_output）：定性、定量或结构解析

2. **选择核心技术平台**
   - **核磁共振**（NMR）
     - 适用场景：需无损检测、样本珍贵、需结构信息
     - 常用谱型：¹H-NMR、¹³C-NMR、³¹P-NMR
     - 优势：无需衍生化、可重复性强
   
   - **质谱**（MS）
     - 适用场景：需高灵敏度定性或定量分析
     - 输出：代谢产物谱（基于质荷比 m/z）
   
   - **色谱技术**
     - 包括：GC、LC、HPLC、UPLC、TLC
     - 辅助技术：毛细管电泳（CE）、FTIR、FT-Raman

3. **优先考虑联用技术**（推荐）
   - **GC-MS**：适用于挥发性小分子代谢物
   - **LC-MS / HPLC-MS**：适用于极性大、热不稳定分子
   - **UPLC-MS**：适用于高通量筛查

4. **输出推荐方案**
   - 综合代谢物性质与分析目标，输出具体技术组合（例如：“LC-MS用于极性代谢物定量”）

> 注意：单一技术难以覆盖全部代谢物多样性，应根据实际需求灵活组合。
