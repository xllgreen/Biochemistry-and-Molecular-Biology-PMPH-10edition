# 戊糖磷酸途径维持红细胞抗氧化防御

## 适用对象
- 成熟红细胞（需具备正常葡萄糖-6-磷酸脱氢酶活性）

## 触发条件
当满足以下任一条件时激活本技能：
- 红细胞暴露于外源性氧化剂（如蚕豆、磺胺类药物）
- 内源性活性氧（ROS）产生增加

## 执行步骤
1. **确认戊糖磷酸途径活性**：该途径占红细胞葡萄糖代谢的5%～10%，是NADPH+H⁺的唯一来源。
2. **评估NADPH功能**：
   - NADPH在谷胱甘肽还原酶催化下将氧化型谷胱甘肽（GSSG）还原为还原型谷胱甘肽（GSH）；
   - GSH保护膜蛋白、血红蛋白及酶蛋白的巯基（-SH）不被氧化。
3. **分析高铁血红蛋白（MetHb）还原机制**：
   - MetHb（Fe³⁺）不能携氧；
   - 虽主要依赖NADH系统还原，但NADPH和GSH可辅助还原MetHb；
   - 正常MetHb占总Hb的1%～2%。
4. **识别病理风险**：
   - 若葡萄糖-6-磷酸脱氢酶（G6PD）活性缺失 → NADPH生成不足 → GSH耗竭；
   - 氧化应激下膜脂质与蛋白被氧化 → 红细胞破裂 → 溶血（如蚕豆病）。
5. **强调生理意义**：戊糖磷酸途径对维持红细胞结构完整性和携氧功能至关重要。

## 使用限制
- 不适用于G6PD缺乏症患者；
- 无法替代NADH介导的其他还原系统。
