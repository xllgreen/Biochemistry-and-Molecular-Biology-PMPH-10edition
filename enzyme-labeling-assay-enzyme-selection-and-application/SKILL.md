# 酶标记测定法中标记酶的选择与应用

## 何时使用
当满足以下条件时使用本技能：
- 需要检测抗原或抗体
- 存在特异性抗原-抗体结合反应
- 需避免使用放射性核素，转而采用安全、稳定的酶标记方法
- 应用于ELISA或其他酶联免疫测定平台

## 执行步骤
1. **选择合适的标记酶**，常用类型包括：
   - 辣根过氧化物酶（HRP）
   - 碱性磷酸酶（ALP）
   - 葡萄糖氧化酶（GOD）
   - β-D-半乳糖苷酶（β-galactosidase）
2. **制备酶-抗体复合物**：将所选标记酶通过化学偶联方法与特异性抗体连接。
3. **结合目标分子**：利用抗原-抗体特异性结合，使酶-抗体复合物与待测抗原（或抗体）形成复合物。
4. **加入对应底物**：添加该酶特异性的底物，酶催化反应生成可检测信号（如显色、荧光或化学发光）。
5. **定量分析**：测量信号强度，其与目标抗原/抗体浓度成正比，用于定量。

## 注意事项
- 不适用于非特异性结合体系
- 不适用于优先使用放射性核素标记的场景
- 所选酶必须具备高催化活性和稳定信号输出能力

完成上述流程后，即可获得安全、灵敏且可重复的微量生物分子检测结果。
