# 真核生物RNA聚合酶选择与特性判断

## 触发条件
当任务涉及以下任一情形时启用本技能：
- 需要根据目标RNA类型确定对应的真核RNA聚合酶
- 需要预测RNA聚合酶的亚细胞定位（核仁 vs. 核内非核仁区域）
- 需要评估RNA聚合酶对抑制剂α-鹅膏蕈碱的敏感性
- 需要确认是否含有羧基末端结构域（CTD）等关键特征

> **注意**：本技能仅适用于真核细胞核内转录，不适用于原核生物或线粒体/叶绿体中的RNA聚合酶。

## 执行步骤

1. **识别目标RNA类型**
   - 若为 **45S rRNA前体**（最终加工为28S、5.8S、18S rRNA）→ 使用 **RNA聚合酶I**
   - 若为 **前体mRNA、lncRNA、miRNA 或 piRNA** → 使用 **RNA聚合酶II**
   - 若为 **tRNA、5S rRNA 或部分snRNA** → 使用 **RNA聚合酶III**

2. **确定亚细胞定位**
   - RNA pol I：**核仁**
   - RNA pol II 和 III：**细胞核内（核仁外）**

3. **判断对α-鹅膏蕈碱的敏感性**
   - RNA pol I：**耐受（不敏感）**
   - RNA pol II：**极敏感**
   - RNA pol III：**中度敏感**

4. **检查关键结构特征**
   - 仅 **RNA pol II** 含有 **羧基末端结构域（CTD）**，其七肽重复序列为 Tyr-Ser-Pro-Thr-Ser-Pro-Ser
     - 酵母约27次重复，哺乳动物约52次重复
     - CTD的Ser/Tyr残基磷酸化对转录起始至关重要

5. **确认共同特征（用于验证）**
   - 三者均为含12个以上亚基的多亚基复合物
   - 均含两个大亚基（160–220kD 和 128–150kD），分别与大肠杆菌β'和β亚基同源
   - 各自识别不同类型的启动子（I类、II类、III类）

## 输出模板
返回包含以下信息的结构化结果：
- 所选RNA聚合酶类型
- 转录产物
- 亚细胞定位
- α-鹅膏蕈碱敏感性
- 是否含CTD
- 关键功能备注
