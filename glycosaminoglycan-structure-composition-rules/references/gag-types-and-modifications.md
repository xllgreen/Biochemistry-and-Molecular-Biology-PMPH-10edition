# 六种主要糖胺聚糖的详细组成与修饰特征

## 1. 硫酸软骨素（Chondroitin Sulfate）
- **二糖组成**：N-乙酰半乳糖胺 + 葡萄糖醛酸
- **常见硫酸化位点**：N-乙酰半乳糖胺的C₄和/或C₆位
- **功能关联**：软骨、肌腱等结缔组织的主要成分

## 2. 硫酸皮肤素（Dermatan Sulfate）
- **二糖组成**：N-乙酰半乳糖胺 + 艾杜糖醛酸（部分由葡萄糖醛酸经差向异构酶转化而来）
- **硫酸化位点**：类似硫酸软骨素，但艾杜糖醛酸可于C₂位硫酸化
- **分布**：皮肤、血管壁

## 3. 硫酸角质素（Keratan Sulfate）
- **二糖组成**：半乳糖 + N-乙酰葡萄糖胺（注意：不含己糖醛酸）
- **连接方式**：可通过O-连接（丝氨酸）或N-连接（天冬酰胺）附着于核心蛋白
- **硫酸化**：N-乙酰葡萄糖胺的C₆位及半乳糖的C₆位

## 4. 透明质酸（Hyaluronic Acid / Hyaluronan）
- **二糖组成**：葡萄糖醛酸 + N-乙酰葡萄糖胺
- **关键特征**：**不含硫酸基团**，由质膜上的合酶直接合成（非高尔基体）
- **功能**：润滑、保水、细胞迁移信号

## 5. 肝素（Heparin）
- **二糖组成**：高度修饰的葡萄糖胺（N-硫酸化、C₆-O-硫酸化） + 艾杜糖醛酸（C₂-O-硫酸化）
- **合成特点**：在肥大细胞中合成，具有最强负电荷密度
- **功能**：抗凝血活性

## 6. 硫酸类肝素（Heparan Sulfate）
- **结构**：与肝素相似，但硫酸化程度较低、序列更复杂
- **分布**：几乎所有细胞表面及基底膜
- **功能**：生长因子结合、信号转导调控

## 合成机制补充说明
- **起始**：在核心蛋白的丝氨酸（O-连接）或天冬酰胺（N-连接）上启动
- **延伸**：由特异性糖基转移酶以UDP-单糖为供体逐个添加单糖
- **修饰**：随后进行差向异构化（如葡萄糖醛酸→艾杜糖醛酸）、N-脱乙酰化/N-硫酸化、O-硫酸化等
- **硫酸供体**：3'-磷酸腺苷5'-磷酰硫酸（PAPS）
- **氨基来源**：谷氨酰胺提供氮源用于己糖胺合成

## 约束条件重申
- 不适用于非糖胺聚糖（如淀粉、纤维素、糖原）
- 透明质酸虽属GAG，但合成路径与其他GAG不同（不依赖高尔基体糖基转移酶级联）
