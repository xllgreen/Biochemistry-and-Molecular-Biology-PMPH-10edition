# 糖胺聚糖结构组成规则

## 何时使用
- 分析生物样本中酸性多糖的结构
- 研究蛋白聚糖的聚糖链组成
- 鉴别六种主要糖胺聚糖类型（硫酸软骨素、硫酸皮肤素、硫酸角质素、透明质酸、肝素、硫酸类肝素）
- 预测糖胺聚糖的电荷特性或功能潜力

## 执行步骤
1. **确认基本结构单元**：所有糖胺聚糖均由重复的二糖单位构成，每个二糖包含一个己糖醛酸（葡萄糖醛酸或艾杜糖醛酸）和一个己糖胺（如N-乙酰半乳糖胺、N-乙酰葡萄糖胺或葡萄糖胺）。
2. **识别具体GAG类型**：根据单糖组合与修饰特征判断所属类别（参见参考文件 `gag-types-and-modifications.md`）。
3. **检查硫酸化状态**：除透明质酸外，所有糖胺聚糖均含硫酸基团；硫酸化位点（如C₄、C₆、N位、C₂）决定其电荷密度与生物学功能。
4. **验证合成环境**：确保分析对象来源于内质网或高尔基体中的酶促合成过程，且依赖UDP-单糖供体与PAPS（3'-磷酸腺苷5'-磷酰硫酸）作为硫酸供体。
5. **排除非适用对象**：不适用于淀粉、糖原等非糖胺聚糖类多糖。

> 注意：透明质酸虽为糖胺聚糖，但不含硫酸基团，是唯一例外。
