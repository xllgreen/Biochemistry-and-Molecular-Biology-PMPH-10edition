# 大肠埃希菌表达体系适用性评估

## 触发条件
当面临以下任一场景时，启动本技能：
- 制定重组蛋白生产策略
- 进行表达系统选择决策
- 开发生物制药工艺

且尚未确定是否采用大肠埃希菌作为表达宿主。

## 执行步骤

### 第一步：确认前提条件
确保已明确以下信息：
- 目标蛋白来源（原核/真核）及其是否需要翻译后修饰（如糖基化）
- 下游纯化方法及功能验证需求

### 第二步：评估E. coli体系的优点
检查是否满足以下优势条件：
1. **高表达水平**：适用于需毫克至克级产量的蛋白
2. **操作简便、成本低**：适合标准化培养和大规模发酵
3. **遗传操作成熟**：载体构建和转化效率高
4. **机制清晰**：便于基于已知调控元件进行理性设计

### 第三步：识别关键限制
判断是否存在以下任一限制：
1. **无法处理内含子**：仅能表达cDNA序列
2. **缺乏翻译后修饰能力**：不能进行糖基化等真核特异性修饰
3. **易形成包涵体**：尤其对真核蛋白，可能导致无活性聚集体
4. **可溶性表达困难**：疏水性或结构复杂蛋白难以获得功能性可溶产物

### 第四步：做出适用性判断
根据上述分析，输出以下结论之一：
- **推荐使用E. coli**：若目标蛋白为原核来源、无需复杂修饰、且对可溶性有优化方案
- **建议改用真核表达体系（如酵母、昆虫、哺乳动物细胞）**：若目标蛋白需糖基化、正确折叠依赖伴侣蛋白、或为分泌型/膜蛋白

## 关键变量参考
- **表达水平**：E. coli通常为“高”
- **是否形成包涵体**：真核蛋白常为“是”
- **是否需翻译后修饰**：若为“是”，则E. coli不适用
