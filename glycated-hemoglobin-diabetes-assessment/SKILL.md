# 糖化血红蛋白用于糖尿病诊治评估

## 何时使用
- 需要评估糖尿病患者的长期血糖控制效果
- 辅助诊断糖尿病（尤其当空腹或随机血糖结果不明确时）
- 比较不同治疗方案对血糖控制的稳定性影响

## 如何执行
1. **确认适用条件**：确保患者无溶血性贫血、近期输血史或血红蛋白变异体（如HbS、HbC），且红细胞寿命正常。
2. **获取GHB检测值**：以百分比（%）形式表示糖化血红蛋白占总血红蛋白的比例。
3. **解读结果**：
   - 若 GHB ≥ 6.5%，提示符合糖尿病诊断标准；
   - 若 GHB < 6.5%，通常视为血糖控制正常；
   - 对已确诊糖尿病患者，治疗目标为将GHB控制在7%以下。
4. **说明优势**：强调GHB不受短期饮食、运动或应激干扰，反映长期血糖平均水平，优于单次血糖检测。
5. **提醒注意事项**：若存在可能干扰检测的血红蛋白变异体，建议结合其他指标（如空腹血糖、OGTT）综合判断。

> 提示：本技能仅用于临床参考，最终诊断需结合患者整体临床表现及其他实验室检查。
