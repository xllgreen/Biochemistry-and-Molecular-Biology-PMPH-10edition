# 肝脏生物转化通路分析

## 何时使用
- 需判断某药物或毒素是否经肝脏代谢
- 分析代谢产物是否更具毒性（如致癌物活化）
- 评估肝功能障碍对药物清除的影响
- 设计给药方案时考虑代谢酶诱导/抑制

## 如何执行
1. **确认前提条件**：
   - 肝细胞功能正常
   - 辅因子充足（如NADPH用于第一相，UDP-葡萄糖醛酸用于第二相）
2. **执行第一相反应（功能化）**：
   - **氧化**：主要由细胞色素P450催化，引入-OH、-COOH等极性基团
   - **还原**：如硝基→氨基
   - **水解**：酯/酰胺→酸+醇/胺
3. **执行第二相反应（结合）**：
   - **葡萄糖醛酸结合**（最常见）
   - **硫酸结合**
   - **乙酰化、甲基化、谷胱甘肽或甘氨酸结合**
4. **评估三大特点**：
   - **连续性**：多数物质需经历两相反应
   - **多样性**：同一物质可走多条代谢路径
   - **双重性**：通常解毒，但可能致毒（如黄曲霉素B1→环氧化物）
5. **输出结论**：
   - 代谢产物水溶性是否增加
   - 是否需监测毒性或调整剂量
   - 排泄途径（尿液或胆汁）

> 注意：严重肝病患者此通路受损，应避免依赖肝代谢的药物。
