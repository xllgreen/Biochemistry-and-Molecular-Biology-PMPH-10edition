# 水的反常热胀冷缩特性

## 何时使用
- 当水温低于4°C且需解释其密度变化趋势时
- 需说明冰为何能浮于水面
- 分析冬季湖泊结冰后水下生物如何存活
- 探讨水分子氢键结构对体积的影响

## 执行步骤
1. 确认水温处于0°C至100°C范围内，且环境为常规压力（非超临界状态）。
2. 判断当前水温是否低于4°C：
   - 若高于4°C，水遵循常规热胀冷缩规律（温度↑ → 体积↑ → 密度↓）。
   - 若低于4°C，应用反常膨胀规律（温度↓ → 体积↑ → 密度↓）。
3. 在0°C时，水结冰，体积增大约1/9，密度降至约0.9 g/cm³，小于液态水。
4. 基于冰浮于水面的事实，推导其生态意义：表面冰层隔绝冷空气，保护下方液态水中的生物。
5. 如需深入解释，引用水分子在低温下通过氢键形成有序四面体结构导致体积膨胀的机制。

> 注意：本技能不适用于重水（D₂O）或超临界水状态（647 K, 22.064 MPa）。
