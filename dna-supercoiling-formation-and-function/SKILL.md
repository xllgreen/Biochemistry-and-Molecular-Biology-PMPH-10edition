# DNA超螺旋结构的形成与功能

## 何时使用
- 解释原核或真核DNA为何呈现超螺旋构象
- 分析拓扑异构酶如何调控DNA拓扑状态
- 判断负超螺旋是否能在特定条件下稳定存在
- 阐述超螺旋如何促进DNA局部解链以支持复制或转录

## 执行步骤
1. **确认前提条件**：DNA双链具有柔韧性、存在内部弯曲应力、有拓扑异构酶参与。
2. **识别适用对象**：仅适用于原核生物环状DNA、真核生物线性DNA（需与蛋白质结合）或拓扑异构酶作用下的DNA。
3. **判断超螺旋类型**：
   - 若盘绕方向与DNA双螺旋相同 → 正超螺旋
   - 若相反 → 负超螺旋
4. **验证稳定性条件**：负超螺旋仅在以下情况稳定：
   - 封闭环状DNA分子（如质粒、细菌染色体）
   - 与组蛋白或其他DNA结合蛋白结合的状态
5. **评估生物学功能**：负超螺旋可降低局部双链稳定性，促进解链，从而增强复制起始、转录启动等过程效率。
6. **量化参考值**：在大肠杆菌中，平均每200个碱基对形成一个负超螺旋。

> 注意：不适用于完全刚性DNA模型或无封闭结构/蛋白质结合的线性DNA。
