# mRNA 5'-帽子结构的功能与组成

## 何时使用
- 解释真核mRNA为何比原核更稳定
- 分析翻译起始效率的调控机制
- 研究mRNA出核转运的分子基础
- 探讨去帽酶或帽子类似物的作用

## 执行步骤
1. **确认适用对象**：仅限真核生物成熟mRNA。
2. **识别帽子结构**：由7-甲基鸟苷（m⁷G）通过5'-5'三磷酸桥连接至mRNA 5'端第一个核苷酸。
3. **评估三大功能**：
   - **稳定性**：抵抗5'→3'外切酶降解
   - **核输出**：作为加工完成标志，协助mRNA通过核孔
   - **翻译起始**：被eIF4E识别，促进43S预起始复合物结合
4. **关联转录时机**：帽子在转录早期（约20–30 nt长度时）即被添加，是mRNA成熟的关键步骤。

> 注意：原核mRNA无5'帽子结构；帽子缺失将导致mRNA快速降解和翻译失败。
