# TP53抑癌机制：DNA损伤应答与细胞凋亡调控

## 适用条件
- 细胞类型：人类体细胞、肿瘤细胞或已发生DNA损伤的细胞
- 前提条件：存在DNA损伤（如电离辐射或化学试剂诱导），且TP53基因为野生型（未发生功能丧失性突变）
- 不适用于：TP53基因已突变的细胞，或无DNA损伤的正常稳态细胞

## 执行流程
1. **检测DNA损伤**：确认细胞是否受到电离辐射、化学试剂等导致DNA断裂或碱基损伤。
2. **评估TP53状态**：验证TP53基因为野生型，确保p53蛋白功能完整。
3. **激活p53通路**：
   - p53蛋白表达水平迅速升高（可升高5～100倍以上）；
   - 特定丝氨酸残基被磷酸化修饰，活化p53；
   - 活化的p53从细胞质移位至细胞核。
4. **调控下游靶基因转录**：
   - 激活p21 → 阻止G₁/S期检查点通过 → 细胞停滞于G₁期；
   - 激活GADD45 → 启动DNA修复机制。
5. **决策分支**：
   - 若DNA修复成功 → 恢复细胞周期，继续正常分裂；
   - 若DNA修复失败 → 激活促凋亡基因（如BAX）→ 启动细胞凋亡程序，清除潜在癌变细胞。
6. **输出结果**：执行细胞周期阻滞、DNA修复或细胞凋亡，维持基因组稳定性，防止肿瘤发生。

## 关键变量监控
- **p53蛋白水平**：正常半衰期20～30分钟，DNA损伤后显著升高；
- **DNA损伤类型**：明确损伤源（如辐射、烷化剂等）；
- **靶基因表达状态**：监测p21、GADD45、BAX等关键基因的转录活性。

> 注意：若TP53在DNA结合结构域发生突变，该通路失效，需考虑其他干预策略。
