# 细胞色素P450单加氧酶催化机制

## 何时使用
- 需要预测脂溶性药物、类固醇激素前体、毒物或异源化合物经CYP酶羟化后的产物
- 分析药物代谢路径中CYP酶的作用（如解毒或活化）
- 判断某反应是否属于单加氧酶催化（特征：消耗NADPH + O₂，生成ROH + H₂O）
- 评估CYP同工酶（如CYP3A4、CYP2E1）对特定底物的代谢能力

## 如何执行
1. **确认反应条件**：确保存在底物RH、NADPH+H⁺、O₂及含血红素Fe的Cyt P450。
2. **应用反应通式**：
   ```
   RH + NADPH + H⁺ + O₂ → ROH + NADP⁺ + H₂O
   ```
   其中一个氧原子插入底物形成羟化产物ROH，另一个被还原为水。
3. **识别关键组分**：
   - 细胞色素P450（含血红素辅基）
   - NADPH-细胞色素P450还原酶（含FAD）
4. **判断生物学作用**：
   - 是否参与类固醇激素（皮质醇、性激素）合成
   - 是否介导药物/毒物生物转化（增加水溶性或产生活性中间体）
5. **考虑酶特异性与调控**：
   - 主要代谢家族为CYP1、CYP2、CYP3（CYP3A4占肝CYP总量30%~40%）
   - 注意抑制剂（如奥美拉唑）或诱导剂（如利福平）的影响
