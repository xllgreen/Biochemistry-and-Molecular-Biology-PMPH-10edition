# 氨基酸等电点计算详细示例与参考数据

## 1. 中性氨基酸（如丙氨酸）
- 可解离基团：α-COOH (pK₁ ≈ 2.34)，α-NH₃⁺ (pK₂ ≈ 9.69)
- 兼性离子存在于 pK₁ 与 pK₂ 之间
- pI = (2.34 + 9.69) / 2 = **6.02**

## 2. 酸性氨基酸（如天冬氨酸，Asp）
- 可解离基团：α-COOH (pK₁ ≈ 2.0), 侧链-COOH (pK₂ ≈ 3.9), α-NH₃⁺ (pK₃ ≈ 9.9)
- 电荷变化：+1 → 0 → -1 → -2
- 兼性离子（净电荷=0）存在于 pK₁ 与 pK₂ 之间
- pI = (2.0 + 3.9) / 2 = **2.95**

## 3. 碱性氨基酸（如赖氨酸，Lys）
- 可解离基团：α-COOH (pK₁ ≈ 2.2), α-NH₃⁺ (pK₂ ≈ 9.0), 侧链-NH₃⁺ (pK₃ ≈ 10.5)
- 电荷变化：+2 → +1 → 0 → -1
- 兼性离子（净电荷=0）存在于 pK₂ 与 pK₃ 之间
- pI = (9.0 + 10.5) / 2 = **9.75**

## 4. 关键原则
- pI 总是取**净电荷由正变负过程中经过零点时相邻两个pK的平均值**；
- 必须正确排序pK值并分析各pH区间的净电荷；
- 实验测定的pK值可能因环境（温度、离子强度）略有差异，应优先使用文献值。
