# 氨基酸等电点（pI）计算

## 何时使用
- 需要确定某氨基酸在电中性状态下的pH值；
- 用于预测氨基酸或小肽在电泳、等电聚焦或层析中的行为；
- 已知该氨基酸所有可解离基团的pK值。

## 执行步骤
1. **确认适用对象**：仅适用于标准α-氨基酸或已知pK值的含可解离侧链氨基酸（如天冬氨酸、赖氨酸等），不适用于非α-氨基酸或修饰氨基酸（除非其pK数据明确）。
2. **列出所有可解离基团的pK值**，按从小到大排序。
3. **识别兼性离子（净电荷为零）所处的pH区间**：该状态位于两个相邻pK值之间。
4. **计算pI**：取兼性离子两侧相邻pK值的算术平均值。
   - 对于中性氨基酸（两个可解离基团）：pI = (pK₁ + pK₂) / 2；
   - 对于酸性或碱性氨基酸（三个可解离基团）：先确定净电荷为零的区间，再取该区间两侧pK的平均值。
5. **验证结果合理性**：酸性氨基酸pI通常 < 6，碱性氨基酸pI通常 > 7，中性氨基酸pI接近6。
