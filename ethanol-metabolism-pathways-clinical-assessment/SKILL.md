# 乙醇双重代谢通路临床评估

## 何时使用
- 用户询问饮酒后不适（如面红、心悸）的机制
- 评估抗生素或药物与酒精联用的风险
- 分析胎儿、新生儿或东方人群对酒精的特殊敏感性
- 解释酒精引起的乳酸酸中毒、低血糖或肝损伤
- 判断是否需根据基因型或用药史调整饮酒行为

## 执行步骤

### 1. 判断主导代谢通路
- 若血乙醇浓度 ≤ 8.6 mmol/L：主要通过 **ADH通路**（Km = 2 mmol/L）
- 若血乙醇浓度 > 8.6 mmol/L 或长期饮酒：激活 **CYP2E1通路**（Km = 8.6 mmol/L，可被诱导）

### 2. 评估关键前提条件
- 确认肝细胞完整、NAD⁺/NADPH充足、ALDH活性正常
- 若任一条件缺失，代谢受阻，乙醛堆积风险升高

### 3. 识别高危人群与禁忌
- **胎儿/新生儿**：ADH未表达，禁酒
- **ALDH缺陷者**（尤其东亚人群）：禁酒或严格限酒
- **使用头孢类、咪唑类抗生素期间**：绝对禁酒，防双硫仑反应

### 4. 分析代谢后果
- NADH/NAD⁺比值↑ → 乳酸酸中毒、低血糖
- CYP2E1激活 → 氧化应激、脂质过氧化 → 肝细胞损伤
- 乙醛堆积 → 血管扩张、心动过速、恶心

### 5. 给出临床建议
- 根据基因背景、年龄、用药史判断个体风险
- 提示避免酒精摄入的具体场景
- 必要时建议检测ALDH2基因型
