# 戊糖磷酸途径详细反应与调控

## 氧化阶段（不可逆）
- 葡萄糖-6-磷酸 + NADP⁺ →（葡萄糖-6-磷酸脱氢酶，Mg²⁺依赖）→ 6-磷酸葡萄糖酸内酯 + NADPH + H⁺
- 6-磷酸葡萄糖酸内酯 + H₂O →（内酯酶）→ 6-磷酸葡萄糖酸
- 6-磷酸葡萄糖酸 + NADP⁺ →（6-磷酸葡萄糖酸脱氢酶）→ 核酮糖-5-磷酸 + NADPH + H⁺ + CO₂
- 核酮糖-5-磷酸 ⇌（磷酸戊糖异构酶）核糖-5-磷酸
- 核酮糖-5-磷酸 ⇌（磷酸戊糖差向异构酶）木酮糖-5-磷酸

## 基团转移阶段（可逆）
涉及以下关键反应（以3分子核糖-5-磷酸为起点）：
- 木酮糖-5-磷酸 + 核糖-5-磷酸 ⇌（转酮醇酶）→ 甘油醛-3-磷酸 + 景天庚酮糖-7-磷酸
- 景天庚酮糖-7-磷酸 + 甘油醛-3-磷酸 ⇌（转醛醇酶）→ 赤藓糖-4-磷酸 + 果糖-6-磷酸
- 赤藓糖-4-磷酸 + 木酮糖-5-磷酸 ⇌（转酮醇酶）→ 果糖-6-磷酸 + 甘油醛-3-磷酸

**总反应式**：
3 葡萄糖-6-磷酸 + 6 NADP⁺ + 3 H₂O → 2 果糖-6-磷酸 + 甘油醛-3-磷酸 + 6 NADPH + 6 H⁺ + 3 CO₂

## 调控机制细节
- **葡萄糖-6-磷酸脱氢酶**：
  - 别构抑制剂：NADPH（Ki ≈ 0.1 μM）
  - 别构激活剂：NADP⁺
  - 表达调控：胰岛素诱导肝脏酶蛋白合成；高碳水化合物饮食上调其mRNA
- **组织差异**：
  - 肝脏：氧化阶段主导，满足脂肪酸合成需求
  - 红细胞：依赖NADPH维持谷胱甘肽还原态，防止溶血
  - 肿瘤细胞：通量增强以支持快速增殖所需的核苷酸与还原力
  - 肌组织：因G6PD缺乏，主要通过糖酵解中间产物经基团转移阶段逆向合成核糖-5-磷酸
