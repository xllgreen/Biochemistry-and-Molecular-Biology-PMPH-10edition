## О проекте

Этот проект систематически интегрирует фундаментальные знания биохимии и молекулярной биологии, охватывая **154 ключевых навыка** в 16 категориях.

**Целевая аудитория** : Студенты-медики, биомедицинские исследователи, клиницисты, преподаватели биохимии, специалисты в области фармации и наук о жизни

**Учебник** : PMPH «Биохимия и Молекулярная Биология» 10-е издание (ISBN: 978-7-117-36618-8)

**⚠️ Риск ⚠️** : Данная коллекция составлена на основе содержания учебника только для образовательных целей. Любые клинические или терапевтические решения должны быть проверены квалифицированными медицинскими специалистами.
